# Portfolio — Pierre Grosjean

Portfolio et projets clients exportés depuis Webflow.

## Structure

```
site/          → Portfolio principal
projects/      → Snapshots sites clients
  projet-1/
  projet-2/
```

## Deploy

Hébergé sur Vercel. Chaque push sur `main` déclenche un deploy automatique.

## Usage

1. Exporter le site depuis Webflow (Code Export)
2. Placer les fichiers dans le bon dossier
3. Push sur `main` → deploy auto
